# Як користуватись програмою?

### BudgetMaster - це легкий в використанні калькулятор витрат,призначений для записування своїх витрат, визначення суми витраченої загально та окремо по категоріям.
#
### Запуск програми:

#### Скопіювати репозиторій (https://github.com/lyrck3r/DungeonMasters.git)
#### Відкрити проєкт в IDEA
#### В папці src відкрити main->java->CostCall
#### Запустити main
#
### Після запуску відкривається графічне вікно програми.
#### Головне меню програми містить:
##### 1.Додати витрату;
##### 2.Показати всі витрати;
##### 3.Загальна сума;
##### 4.Сума по категорії;
##### 5.Видалити витрату;
##### 6.Вийти;
##### Поле для введення тексту.
#
### Опис функції "Додати витрати"
#### -Введіть назву витрати
#### -Введіть суму витрати
#### -Введіть категорію витрати
#
### Опис функції "Показати всі витрати
#### -Показує записані витрати, суму, категорію і дату витрати
#
### Опис функції "Загальна сума"
#### -Показує загальну суму всіх витрат
#
### Опис функції "Сума по категорії"
#### -Введіть назву категорії
#### -Показує суму витрат в цій категорії
#
### Опис функції "Видалити витрату"
#### -Показує всі витрати
#### -Введіть номер витрати
#### -Витрату видалено
#
### Опис функції "Вийти"
#### -Завершує програму
